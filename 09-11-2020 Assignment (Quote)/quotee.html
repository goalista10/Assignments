It’s hard enough to find an error in your code when you’re looking for it; its even harder when you’ve ASSUMED your code is ERROR-FREE.

✈✈✈https://dzone.com/articles/quotes-every-software-engineer✈✈✈

When coding, you are using your own logic and understanding in making the code, that's why it's difficult to look for error on your own code because you don't normally always disagree with yourself.

When you assume your code is error free, then you will never find the error because it's like you are denying it doesn't exist in your code.
